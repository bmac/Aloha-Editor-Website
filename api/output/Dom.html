Insert the given DOM Object into the start/end of the given range. The method will find the appropriate place in the ...
Insert the given DOM Object into the start/end of the given range. The method will find the appropriate place in the DOM tree for inserting the given object, and will eventually split elements in between. The given range will be updated if necessary. The updated range will NOT embrace the inserted object, which means that the object is actually inserted before or after the given range (depending on the atEnd parameter)
Parameters:
  • object : jQuery
    object to insert into the DOM
  • range : GENTICS.Utils.RangeObject
    range where to insert the object (at start or end)
  • limit : jQuery
    limiting object(s) of the DOM modification
  • atEnd : boolean
    true when the object shall be inserted at the end, false for insertion at the start (default)
Returns:
  • true
    if the object could be inserted, false if not.

 split( RangeObject range, jQuery limit, boolean atEnd ) : object
Splits a DOM element at the given position up until the limiting object(s), so that it is valid HTML again afterwards...
Splits a DOM element at the given position up until the limiting object(s), so that it is valid HTML again afterwards.
Parameters:
  • range : RangeObject
    Range object that indicates the position of the splitting. This range will be updated, so that it represents the same range as before the split.
  • limit : jQuery
    Limiting node(s) for the split. The limiting node will not be included in the split itself. If no limiting object is set, the document body will be the limiting object.
  • atEnd : boolean
    If set to true, the DOM will be splitted at the end of the range otherwise at the start.
Returns:
  • object
    jQuery object containing the two root DOM objects of the split, true if the DOM did not need to be split or false if the DOM could not be split
