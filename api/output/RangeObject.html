 findMarkup( function comparator, jQuery limit, boolean atEnd ) : DOMObject
Find certain the first occurrence of some markup within the parents of either the start or the end of this range. Th...
Find certain the first occurrence of some markup within the parents of either the start or the end of this range. The markup can be identified by means of a given comparator function. The function will be passed every parent (up to the eventually given limit object, which itself is not considered) to the comparator function as this. When the comparator function returns boolean true, the markup found and finally returned from this function as dom object.
Example for finding an anchor tag at the start of the range up to the active editable object:
range.findMarkup(
function() {
return this.nodeName.toLowerCase() == 'a';
},
jQuery(GENTICS.Aloha.activeEditable.obj)
);
Parameters:
  • comparator : function
    comparator function to find certain markup
  • limit : jQuery
    limit objects for limit the parents taken into consideration
  • atEnd : boolean
    true for searching at the end of the range, false for the start (default: false)
Returns:
  • DOMObject
    the found dom object or false if nothing found.
